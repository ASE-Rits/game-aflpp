# game-aflpp
SDL2製のゲームをAFL++でテストするためのコンテナを作成する基本構成です。

## 想定環境
Windows11(WSLを使用)

## コンテナ作成方法
1. WSLでコンソールを起動し、このプロジェクトがあるフォルダまで移動(mntフォルダを経由してホストにアクセス)
```
cd /mnt/{このプロジェクトまでのパス}
```

2. コンテナを作成(必ずWSLコンソールから実行)
```
docker compose up
```

## 各ファイル、ディレクトリの概要
- Dockerfile, compose.yml
コンテナを作成するための構成ファイルです。コンテナ名や各設定を変更したい場合、これらのファイルを参照してください。

- data
コンテナとホストOS間で中身が共有されるディレクトリです。デフォルトではコンテナ内の`/game/host`ディレクトリと接続されます。テストデータの転送等に使用してください。

- data/bbgame_kit
ブロック崩し用に作成していたツール関連です。あまり参考にならないと思います...

- data/kit
複数インスタンスを同時起動するテストを自動で行うスクリプトが入っています。詳細はkitディレクトリ内のreadmeを参照してください。
